Difference between map, filter, for each and find. Ans: Map is used to iterate over an array for doing some changes and to assign updated array in new array.Filter is used to filter array if criteria and conditions are met.For each is used to iterate over an array without changing anything.And Find is used to find the values from array under some condition.
